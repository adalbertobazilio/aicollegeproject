# Bot Especialista TechFlow Solar ☀️

Este projeto é um chatbot especialista desenvolvido para o trabalho de Prompt Engineering (Gemini API). O bot é restringido a um domínio específico e não alucina sobre informações externas.

## 🚀 Objetivo
Criar um assistente técnico para o modelo fictício "Inversor TF-X100" que segue regras rígidas de atendimento e encerra a conversa após 3 interações.

## 🛠️ Tecnologias
- Python 3.x
- Google GenAI SDK (Gemini 1.5 Flash)
- Google Colab

## 📋 Passo a Passo para Reproduzir
1. Obtenha uma API Key no [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Abra o arquivo `.ipynb` deste repositório no [Google Colab](https://colab.research.google.com/).
3. No Colab, clique no ícone de chave (Secrets) e adicione sua chave com o nome `API_KEY`.
4. Execute as células em ordem (Instalação das bibliotecas e o Código Principal).
5. Interaja com o bot através das 3 perguntas solicitadas no terminal.
